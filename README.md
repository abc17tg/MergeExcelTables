# MergeExcelTables

MergeExcelTables is a lightweight tool written in C# that consolidates data across Excel files. It checks the first sheet in multiple Excel files and merges them into a single Excel file with one consolidated table.

## Features
- **Automated Consolidation**: Combines the first sheets of multiple Excel files into one.
- **Result in One File**: Produces a single Excel file with all data consolidated for easy access.
- **Lightweight and Fast**: Built with simplicity and speed in mind.

## Getting Started
1. Clone this repository to your local machine:
   ```bash
   git clone https://github.com/abc17tg/MergeExcelTables.git
   ```
2. Open the solution file in Visual Studio.
3. Build and run the project to start merging Excel sheets!
   
## Prerequisites
- .NET installed on your system.
- Microsoft Excel files for processing.
